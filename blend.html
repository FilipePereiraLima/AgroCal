🧪 Blend de Fertilizante NPK

Resolve um sistema linear 3×3 para encontrar as quantidades ideais de três fertilizantes (A, B, C) que atendam exatamente a necessidade de Nitrogênio (N), Fósforo (P) e Potássio (K) da cultura.

O sistema tem a forma M · x = b, onde M é a matriz de composição dos fertilizantes, x são as quantidades a determinar e b é a demanda nutricional. Resolvido pela Regra de Cramer.

Sistemas Lineares na Agronomia

A Regra de Cramer resolve sistemas Ax = b usando determinantes: xᵢ = det(Aᵢ) / det(A), onde Aᵢ é a matriz A com a coluna i substituída pelo vetor de demanda b.

Se o determinante de A for zero, o sistema não tem solução única — isso indica que os fertilizantes escolhidos são linearmente dependentes e não conseguem suprir os três nutrientes de forma independente.
